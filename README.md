Contoh Program Java Sederhana

public class Halo {
    public static void main(String[] args) {
        System.out.println("Halo, ini program Java sederhana!");
    }
}

Penjelasan:

Bagian Kode	Arti

public class Halo	Membuat kelas bernama Halo (nama file harus Halo.java)
public static void main(String[] args)	Metode utama tempat program dijalankan
System.out.println(...)	Perintah untuk menampilkan teks ke layar



---

Cara Menjalankan Program

1. Buka aplikasi Notepad / VSCode / IntelliJ / NetBeans (bebas).


2. Tulis kode di atas dan simpan dengan nama:

Halo.java


3. Buka Command Prompt / Terminal, lalu jalankan:

javac Halo.java


4. Kemudian jalankan:

java Halo
